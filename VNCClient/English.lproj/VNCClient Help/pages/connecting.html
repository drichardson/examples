Using the Connection Window

The Connection Window allows you to identify the computer you want to connect to. You will need the IP address of the remote machine. The remote machine must be running a VNC server.

To use the Connection Window to connect to a VNC server.

  1. Select "New Connection..." from the File menu
  2. Enter the address of the VNC server.
  3. Press the Connect button
  4. Observe the remote user’s desktop, looking for interesting activity.
